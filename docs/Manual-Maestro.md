# KYNDO — Manual Maestro

## 1. Propósito
KYNDO es un juego cognitivo–estratégico de cartas basado en objetos reales.
El objetivo es entrenar memoria, razonamiento visual y toma de decisiones.

## 2. Alcance del MVP
El MVP incluye:
- Juego Memory cognitivo (no clásico)
- Progresión básica del jugador
- Visualización en celular y desktop

El MVP NO incluye:
- Tienda
- Monetización
- Social
- Multijugador avanzado

## 3. Regla de oro
Si algo no está documentado en este Manual Maestro, no existe en el MVP.

## 4. Nombre
El proyecto se llama KYNDO.

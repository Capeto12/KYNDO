# KYNDO — Documentación Oficial

Este directorio contiene los documentos canónicos del proyecto KYNDO.
Si algo no está documentado aquí, no existe en el MVP.

## Documentos

1. Manual Maestro  
   Visión, reglas del juego y decisiones estratégicas congeladas.

2. Manual Técnico  
   Reglas internas del sistema, lógica y cálculos.

3. Esquema de Base de Datos  
   Definición de datos, tablas y relaciones.

4. Manual UX/UI Técnico  
   Comportamiento de pantallas, estados y transiciones.

5. Checklist de Endpoints  
   Contrato entre frontend y backend.

## Regla de oro

No se agregan features ni cambios fuera de estos documentos.
